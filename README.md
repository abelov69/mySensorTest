# mySensorTest
Test "mySensor" library mechanics.

Экспериметирую с приемо-передатчиком NRF24L01.
Одну макетку собираю на arduino pro mini + приемо-передатчик NRF24L01 + плата питания для приемо-передатчика + UART-USB переходник на FT232. Эта плата будет подключена к ПК и будет транслировать данные в терминал.


Вторая макетка собрана на arduino nano + экран + приемо-передатчик NRF24L01 + плата питания для приемо-передатчика + плата с кнопками + плата реле + потенциометр. С нее будут посылаться данные от потенциометра и кнопок, а получать на нее будем данные для экрана и реле.

Прошивку для обоих макеток буду делать в одном проекте, переключаться будет через комментирование определения PCmodule в settings.h
Настройки плат хранятся в файле settings.h
